About


Hey! I'm Harshit Verma, a second-year B.Tech CSE student with a growing passion for both technology and design.
I’m currently exploring the world of web development, learning how to build responsive and functional websites using modern tools and frameworks. I'm also diving into areas like AI integration and problem-solving with code — one step at a time.
Alongside my tech journey, I’m actively involved in the creative side of things. I design user interfaces, event posters, and digital assets using tools like Figma, Canva Pro, and Adobe Illustrator. I currently serve as the Graphics Head of the CSWD society at my college, where I lead the visual design for various events and campaigns.
I love combining logic and creativity — and I’m always looking for opportunities to learn, create, and grow.

EcoSnap AI – Smart Waste Classifier

Tech Stack:

HTML, CSS, JavaScript, Flask (Python), Teachable Machine by Google

Description

An AI-powered web application that classifies waste into biodegradable, recyclable, or hazardous using image recognition. Users get intelligent disposal suggestions and location-based recycling center recommendations. Built with a Figma-inspired design, EcoSnap AI combines functionality and awareness to promote eco-conscious actions.

GreenPath – Sustainable Living Platform

Tech Stack:

React.js, Node.js, MongoDB, Google Maps API (planned)

Description

GreenPath is a long-term eco-tech initiative aiming to guide users toward a greener lifestyle. It will provide personalized sustainability tips, eco-habit tracking, and location-based services for eco-friendly resources. Built with scalability in mind, GreenPath combines clean UI, real-time insights, and community-driven features to promote environmental awareness.
